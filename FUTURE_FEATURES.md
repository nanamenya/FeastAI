# Future Features for FeastAI

This document tracks ideas and planned enhancements for future development.

## High Priority

### 1. Cooking Mode
A simplified, fullscreen view optimized for use during actual cooking:
- Display current step(s) in progress with countdown timer
- Show next upcoming step(s) for awareness
- Large, easily readable text for viewing from a distance
- Minimal UI clutter to reduce distractions
- Voice commands support (optional enhancement)

### 2. Quick Timer Integration
- One-tap timer setting for each cooking step
- Integration with device's native timer
- Visual and audio notifications when steps complete
- Ability to set multiple simultaneous timers

### 3. Progressive Step Highlighting
Real-time visual feedback as cooking progresses:
- **Completed steps**: Grayed out with checkmark
- **Active steps**: Highlighted with distinct color/border
- **Upcoming steps**: Normal state with preview
- Optional: Auto-advance based on time or manual confirmation

## Medium Priority

### 4. Save/Load Meal Plans
- Save complete meal configurations (recipes + target time + kitchen setup)
- Name and organize saved meal plans (e.g., "Thanksgiving 2024", "Sunday Roast")
- Quick recall for recurring events
- Share meal plans with family/friends
- Export/import meal plan JSON

### 5. Font Size Controls
- Quick toggle for larger text (accessibility)
- Especially useful when hands are messy or viewing from distance
- Persist preference across sessions
- Consider presets: Small, Medium, Large, Extra Large

### 6. Enhanced Offline Support
Since this is a PWA, ensure full functionality without internet:
- Cache all recipes, schedules, and settings
- Allow creating/editing recipes offline
- Sync changes when connection restored
- Important for kitchens in basements, garages, or areas with poor connectivity

## Low Priority / Nice-to-Have

### 7. Recipe Import from URLs
- Parse recipe websites automatically
- Extract ingredients, steps, timing, appliances
- Use LLM to structure data into FeastAI format

### 8. Smart Scheduling Suggestions
- Suggest optimal target times based on recipe complexity
- Recommend prep time based on historical data
- Warn about overly ambitious schedules

### 9. Multi-Cook Coordination
- Share schedule with multiple devices
- Real-time updates when steps complete
- Assign specific tasks to different people

### 10. Recipe Photo Integration
- Add photos to recipe steps
- Visual reference during cooking
- Before/after comparisons

### 11. Ingredient Shopping List
- Auto-generate shopping list from selected recipes
- Group by grocery store section
- Check off items as purchased

### 12. Nutritional Information
- Display calories, macros per dish
- Dietary restriction filters (vegetarian, gluten-free, etc.)
